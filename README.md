## Text to Handwriting (PDF)
- [X] Convert a text document (.txt file) into a PDF file with the text content handwritten

## Running the program:
- [X] python 3.5 or 3.6
- [X] dependencies from requirements.txt

```
 $ pip install -r requirements.txt
 ```
